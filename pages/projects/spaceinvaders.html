Project Summary

This is a simple Space Invaders game made using pure JavaScript, HTML5, and CSS. This project helped me learn and practice various JavaScript concepts including:

querySelector(): Selecting elements from the DOM.

addEventListener(): Handling user input through events.

Switch cases: Managing different game states.

keyCodes: Detecting player input from the keyboard.

indexOf(): Searching through arrays.

includes(): Checking for specific values in arrays.

classList(): Adding/removing classes for game visuals.

setInterval(): Controlling the game loop timing.

clearInterval(): Stopping game events or actions.

push(): Adding elements to arrays for game objects.

This project helped me enhance my understanding of JavaScript and its core functionalities, allowing me to create an interactive game.

Project Summary

This project is a VR escape room game made using C# in Unity set in a medieval dungeon. Players must solve four interactive puzzles to escape and wake up in their room. The puzzles include:

Keypad Puzzle: Solve the code to unlock the next area.

Statue Puzzle: Arrange statues by height in the correct order.

Door Puzzle: Find the correct key to open the door.

Chess Puzzle: Place the correct chess pieces into a drop zone in the right order.

The game offers an immersive experience that combines problem-solving and exploration in a medieval-themed environment. The objective is to solve all puzzles and escape the dungeon to return to the real world.
